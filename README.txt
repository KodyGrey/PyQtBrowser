Суть проекта заключалась в создании веб браузера для обычного офисного работяги.
Целью браузера самого по себе была просто возможность просматривать веб страницы в виде вкладок.
Также он должен был запускать приложения жизненно необходимые офисному клерку.
И сами эти приложения должны быть часть проекта и подключаться через браузер.
Я захотел использовать sql базу данных для реализации истории браузера, чтобы потом злой начальник мог ее дистанционно проверить или сделать еще что угодно.
